🆘 QUICK TROUBLESHOOTING:

  "Python not found?"
    → Install from https://www.python.org/downloads/
    → Windows: Check "Add Python to PATH"

  "Port 5000 already in use?"
    → Linux/Mac: lsof -i :5000 | kill -9
    → Windows: netstat -ano | findstr :5000 | taskkill

  "Module not found?"
    → Rerun setup: ./setup.sh (or setup.bat)

  "Database locked?"
    → rm stratos.db && ./setup.sh (or setup.bat)
